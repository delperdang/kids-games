# Kids Games Repository

Welcome to the kids games repository!

## Repository Structure

This repository is organized using branches to keep each game completely separate:

- **`main` branch:** This branch contains only this `README.md` file. It serves as the landing page and provides an overview of the repository's structure.
- **Game Branches:** Each branch other than `main` is dedicated to a single, specific game. If you want to view, play, or edit a particular game, simply check out the corresponding branch.

### Existing Games

Currently available game branches include:
- `wizard-dash`

### How to access a game

To play or view a specific game, switch to its branch. For example, to check out `wizard-dash`, run:

```bash
git checkout wizard-dash
```
